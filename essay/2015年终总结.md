# 2015年的感悟

在即将过去的这一年里，经历了很多，也学会了很多。

无论何时何地，都不要停止学习、停止努力、停止进步。

己所不欲勿施于人！自己要做超人，但每个人对待生活的态度不一样，不是所有人都想做超人。

你不是一个人在战斗，做任何事情的时候，要首先考虑团队如何做，而不是一个人单打独斗。

学无止境，每个人都由自己长处，要向周围的人学习。

有些时候，参与一些交流，比一个人闷着看书要更好。

我是程序员，Just code it！
